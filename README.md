# 🚀 Meeting Insights Generator

An AI-powered web application that converts meeting text or audio into structured insights like **summary, key points, and action items**.


✨ Features

 📝 Text-based meeting analysis
 🎤 Audio upload & recording support
 🤖 AI-generated insights (Groq - LLaMA 3.1)
 📊 Clean summary, key points, action items
 🗂️ Meeting history with search
 📅 Date & time tracking
 📄 Download reports as PDF
 🔐 User authentication (Login/Register)



 🛠️ Tech Stack

 **Frontend:** HTML, CSS, JavaScript
 **Backend:** Flask (Python)
 **AI:** Groq API (LLaMA 3.1)
 **Speech-to-Text:** Whisper
 **Database:** SQLite
 **Authentication:** Flask-Login
 **PDF Generation:** ReportLab
